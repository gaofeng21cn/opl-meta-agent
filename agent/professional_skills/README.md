# OMA professional methods

These methods are internal to OMA Stage execution. They do not project additional public actions.

- `oma-design-basis-architecture`: mission, design evidence, transfer, and Stage architecture.
- `oma-target-assessment-eval-design`: takeover assessment and protected-boundary-aware EvalSpec design.
- `oma-evidence-diagnosis`: evidence-bound root-cause analysis.
- `oma-evolution-proposal`: complete next-blueprint and semantic-diff authoring.
